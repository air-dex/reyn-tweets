Enumeration Type Documentation

enum CoreResult

Unified error for all the possible ends of all treatments.

Enumerator:
INVALID_ISSUE 

Issue for fake process results.

UNKNOWN_PROBLEM 

The request ends with an unknown problem.

PARSE_ERROR 

There were an error while parsing results.

NETWORK_CALL 

There were an error while calling the network.

NO_MORE_DATA 

There was no more data to return.

Corresponding Twitter return code is 304.

BAD_REQUEST 

The request was invalid.

Corresponding Twitter return code is 400.

REFUSED_REQUEST 

Twitter did not treat the request.

Corresponding Twitter return code is 403.

RESOURCE_NOT_FOUND 

The URI of the request or the requested resource is invalid.

Corresponding Twitter return code is 404.

INVALID_SEARCH 

The search format is invalid.

Corresponding Twitter return code is 406.

RATE_LIMITED 

Rate limit was hit.

Corresponding Twitter return code is 420.

TWITTER_DOWN 

Twitter got problems.

Corresponding Twitter return code looks like 5xx.

LOAD_CONFIGURATION_SUCCESSFUL 

The launching was successful.

CONFIGURATION_FILE_UNKNOWN 

The configuration file does not exist.

CONFIGURATION_FILE_NOT_OPEN 

The configuration file cannot be opened.

LOADING_CONFIGURATION_ERROR 

The configuration cannot be launched.

SAVE_SUCCESSFUL 

The launching was successful.

EXPECTED_KEY 

The launching was successful.

TOKENS_OK 

Credentials were right.

TOKENS_NOT_AUTHORIZED 

Credentials were wrong.

Corresponds to the Twitter return code 401 too.

NO_TOKENS 

Not obtained tokens (OAuth Authentication Flow).

AUTHORIZED 

The application is authorized.

DENIED 

The application is denied.

POST_AUTHORIZING_FAILED 

Unexpected redirection during POST authorizing.

UNSHORTENABLE_MESSAGE 

While using TwitLonger, if the tweet cannot be shortened.

MESSAGE_NOT_UPDATED 

While using TwitLonger, if the long message in TwitLonger cannot be updated with the final tweet ID.

LAUNCH_SUCCESSFUL 

LaunchingProcess ended successfully.

AUTHENTICATION_REQUIRED 

An authentication to Twitter is required.

ALLOW_SUCCESSFUL 

AllowProcess ended successfully.

FAVORITE_SUCCESSFUL 

FavoriteProcess ended successfully.

TIMELINE_RETRIEVED 

After retrieving a timeline.

TWEET_DELETED 

After deleting a tweet.

TWEET_POSTED 

After updating a tweet.

TWEET_RETWEETED 

After retweeting a tweet.

WRONG_USER 

The user was wrong.

TWEET_UNDESTROYABLE 

If a tweet cannot be deleted.
