QDomElement TwitLongerRequester::getInfoElement ( QDomElement  xmlRoot) [protected, pure virtual]

Getting the XML element with all the necessary informations.

TwitLonger replies are XML streams whose root node called "twitlonger". This node or one of its children contains a node with all the needed information. For example, the TwitLonger reply of the request which posts the message to TwitLonger got all what the application needs in a node called "post" which is the children of the root "twitlonger" node. That is this "post" node that this method will return because its children contain crucial informations just like the message ID and the shortened tweet.

Parameters:
xmlRootRoot of the XML reply.
Returns:
The XML element whose children contained the informations.

Implemented in PostToTwitLongerRequester, and SendIDToTwitLongerRequester.
