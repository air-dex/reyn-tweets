Enumeration Type Documentation

enum ErrorType

Enumeration which indicates if something bad happened during the request. It can takes three values depending on what happened. See the enum values for more details.

Enumerator:
NO_ERROR 

Code indicating that there was no error.

INVALID_RESULT 

Code for an invalid request result.

API_CALL 

Code indicating that there were an error while calling the Twitter API.

SERVICE_ERRORS 

Code indicating that the service (ex. Twitter) has returned errors.

QJSON_PARSING 

Code indicating that there was an error while parsing results with QJson.

OAUTH_PARSING 

Code indicating that there was an error while parsing results under the form arg1=val1&arg2=val2&...&argN=valN.

HTML_PARSING 

Code indicating that there was an error while parsing HTML.

XML_PARSING 

Code indicating that there was an error while parsing XML.
